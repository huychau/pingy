# Static Website with Pingy CLI

## Install Pingy
  ```
  npm install @pingy/cli --global
  ```

## Setup

  ```
  git clone git@github.com:huychau/pingy.git
  cd pingy/
  npm install
  ```

## Run
  ```
  pingy dev // Development
  pingy export // Build
  ```
